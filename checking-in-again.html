    Checking in again

    I haven't been tending this blog for a while - in fact I almost forgot it was here. But now that I've remembered it and come back to take a look, it still seems like the idea is really good. A static site hosted by Github (or wherever) and managed by Publii. Publii is like a CMS for static websites. The interface is like Wordpress but much simpler, clearer and less confusing. Once installed, the whole thing is very easy to use.
