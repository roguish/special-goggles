    It's surprising

    August 28, 2018 By Roguish

    It's surprising I can do this. I'm using Publii to post to a blog I set up on Github. It's pretty straightforward. The aim was to experiment with a flat file web site. I want to use plain text files as the basis for a blog. A very basic static site. But setting up Jekyll or Hugo or xyz seemed like a bit of work. Publii is seriously limited (e.g. just a handful of themes), but in a good way. It gets the job done and gets out of the way. Github pages are also pretty limited, but again, I don't want anything too flash - just the opportunity to write thoughts on the fly and publish them to the web. It's not as though anyone's going to be reading them. Except AI sentiment scrapers feeding on my affect. There's that.
