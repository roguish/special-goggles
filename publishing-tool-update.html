    Publishing tool update

    I’m using Publii to publish this website on Github.

    Now there’s a new Markdown editor, in addition to the older WYSIWYG editor. So, having updated the app, I’m:

    • using it
    • testing it
    • liking it.

    Benefits

    1. Very simple.
    2. It just works.
    3. Also, the idea is that the editor gets out of your way. Nothing to distract from just writing.
    4. You can keep your fingers on the keyboard and add simple styling using Markdown conventions such as bold text. There are keyboard shortcuts for just about everything, as listed on the SimpleMDE ReadMe page
    5. It’s just a simple Markdown editor, though, so extensions such as strikethrough probably don’t work. Update: They do. Perhaps all Github Flavoured Markdown conventions will work. Need to check. Update: Looks like SimpleMDE uses GFM.

    Using Github Flavoured Markdown

    There are some extensions to basic Markdown, or Commonmark, such as:

    • Strikethrough (`~~)
    • Checkboxes like the ones you see here (or not!!)
    • Autolinks such as: www.commonmark.org
